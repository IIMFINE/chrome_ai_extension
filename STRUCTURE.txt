文件说明：

【核心功能文件】
manifest.json       - 定义扩展权限、图标、脚本等配置
popup.html         - 用户界面布局（设置、聊天窗口）
popup.css          - 界面样式（渐变色、动画、布局）
popup.js           - 前端逻辑（消息处理、API 调用、状态管理）
content.js         - 提取网页内容（标题、URL、文本）
background.js      - 后台服务（API 请求处理）

【图标文件】
*.png              - Chrome 扩展所需的 PNG 格式图标
*.svg              - 可编辑的矢量图标源文件

【工具脚本】
generate_icons.sh  - 使用 librsvg/ImageMagick 转换 SVG 到 PNG
generate_icons.py  - 使用 Python Pillow 生成简单图标

【文档文件】
README.md          - 功能介绍、使用方法、配置说明
INSTALL.md         - 详细的安装步骤和配置教程
QUICKSTART.md      - 1 分钟快速开始指南
PROJECT_SUMMARY.md - 项目完成情况总结
config.example.json- API 配置示例

---

使用流程：
1. 用户点击扩展图标
2. Content Script 读取当前网页内容
3. 用户输入问题
4. Popup 构建完整消息（系统提示 + 网页内容 + 问题）
5. Background Service 调用 AI API
6. 返回并显示 AI 回复
7. 保持对话上下文
